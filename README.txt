####Deskripsi Program #####
Program ini bertujuan untuk mencari titik-titik Convex Hull untuk Visualisasi Tes Linear Spearability Dataset

####Requirement Program####
Python, Jupyter Notebook, Module Library numpy,pandas dan matplotlib
Module Library sklearn jika ingin menggunakan dataset dari sklearn

###Cara Menggunakan Program####
###Dengan file main.ipnyb####
1. Pastikan seluruh file di dalam folder src berada dalam satu folder
2. Buka file main.ipnyb dengan menggunakan Jupyter Notebook
3. Jalankan semua cell dari awal hingga akhir untuk mengetahui grafik Convex Hull
4. Jika ingin melakukan pengeditan pada file main.ipnyb bisa dengan Copy Paste cell ke 3 untuk loading dataset baru dan Copy Paste cell 5 untuk menampilkan grafik
###Dengan file .ipnyb atau .py lainnya
1. Pastikan seluruh file di dalam folder src kecuali main.ipnyb berada dalam satu folder
2. Buka file .ipnyb dengan menggunakan Jupyter Notebook
3. tuliskan "from MainConvexHull import *" di code atau cell bagian atas sendiri
4. Load dataset pada umumnya
5. Sebelum mengeplot Convex Hull pastikan sudah mengimport matplotlib.pyplot
6. Pastikan juga dataset yang dimasukkan ke dalam objek ConvexHull hanya berisi dua atribut
7. Jalankan semua cell dari awal untuk mengetahui grafik Convex Hull
